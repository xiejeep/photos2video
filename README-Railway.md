## 🚀 部署步骤

### 1️⃣ 检查准备状态
```bash
npm run prepare-deploy
```

### 2️⃣ 推送到GitHub
```bash
git add .
git commit -m "准备Railway部署"
git push origin main
```

### 3️⃣ Railway部署
1. 访问 [railway.app](https://railway.app)
2. 点击 **"Start a New Project"**
3. 选择 **"Deploy from GitHub repo"**
4. 选择你的仓库
5. 等待自动部署完成 ⏱️
