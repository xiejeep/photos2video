🔧 使用说明

  1. 将字体文件放在 public/fonts/ 目录下
  2. 在 fonts.css 中定义字体
  3. 刷新此页面检查字体是否正确显示
  4. 如果字体显示正常，就可以在应用中使用了

提示：如果某个字体没有正确显示，请检查：

  • 字体文件路径是否正确
  • CSS 中的 font-family 名称是否匹配
  • 字体文件是否完整
  • 浏览器控制台是否有错误信息
